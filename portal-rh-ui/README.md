# Portal RH UI (visión)

UI estática para demo del portal RH con módulos operativos (organización, personal, liquidación, vacaciones, tiempos, integraciones).

## Uso
1. Levantá el stack RRHH + `nucleuswf-service` + `integration-hub-service` + `portal-bff-service` (opcional).
2. Abrí `index.html`.
3. Click en "Login demo" y luego "Refrescar datos".

Si configurás `Portal BFF`, el portal usa `/api/rh/v1/*` en lugar de los servicios directos.
