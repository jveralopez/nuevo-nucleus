# Plan 100% operativo · Nucleus RH Next

## Objetivo
Completar todos los modulos blueprint y dejar el sistema productivo con operacion real (integraciones, datos migrados, UI completa, QA y monitoreo).

## Criterios de completitud por modulo
- API completa (CRUD + flujos + validaciones + auditoria).
- UI operativa (ABM, reportes, flujos, permisos).
- Workflow integrado (cuando aplica).
- Integraciones externas reales (cuando aplica).
- Casos de uso/prueba documentados y ejecutables.

## Fases (propuesta)

## Avance actual
- Fase 1: 75% (Configuracion, Organizacion y Personal completados; Tiempos Trabajados pendiente).

### Fase 1 · Fundaciones y dependencias
**Objetivo:** habilitar los modulos que son base de datos y catalogos para el resto.
- Configuracion Avanzada (catalogos, diccionarios, parametros, feature flags).
- Organizacion (empresas, unidades, posiciones, centros de costo, organigramas).
- Personal (legajos completos, familiares, documentos, solicitudes con aprobacion).
- Tiempos Trabajados (fichadas, turnos, licencias, exporte a Liquidacion).

### Fase 2 · Operaciones core RRHH
**Objetivo:** completar flujos core en RRHH.
- Vacaciones (saldo, aprobacion completa, integracion con Tiempos y Liquidacion).
- Reclamos (workflow completo, SLA, adjuntos).
- Sanciones (workflow disciplinario).
- Medicina Laboral (licencias medicas, aptos, reportes).

### Fase 3 · Talento y desarrollo
**Objetivo:** habilitar gestion de talento end-to-end.
- Seleccion/WebCV.
- Evaluacion de desempeño.
- Capacitacion.
- Carrera/Sucesion.
- Clima/Encuestas.

### Fase 4 · Finanzas y compliance
**Objetivo:** cerrar el circuito financiero y legal.
- Tesoreria (adelantos, pagos, conciliaciones).
- Presupuesto (headcount y costos).
- Beneficios.
- Accidentabilidad.
- Seguridad y Control de Visitas.

### Fase 5 · Analytics e integraciones finales
**Objetivo:** data platform y reporting unificado.
- Analytics central (ETL/ELT, dashboards, permisos).
- Integraciones externas finales (bancos, AFIP, ERP, ATS).

## Entregables por fase
- Servicios + UI por modulo.
- Migraciones y datos de prueba.
- Casos de uso/prueba actualizados.
- E2E por modulo.
- Reporte de validacion.
