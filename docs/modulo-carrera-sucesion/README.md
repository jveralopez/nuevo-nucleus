# Módulo Planes de Carrera & Sucesión · Blueprint

## Objetivo
Modernizar el módulo de planes de carrera/sucesión (parte de Talent Management) para gestionar trayectorias, sucesores críticos, gaps y planes de desarrollo.

## Visión actual (23.01)
- Clases y reportes nominales (no detallados) en Application.xml (Evaluacion, Carrera). Probablemente usa datos de Legajos, Evaluación y Organización.

## Propuesta moderna
1. **Carrera/Sucesión Service**: API para trayectorias, mapas de sucesión, pools de talento y planes de acción.
2. **UI Talent**: matrices 9-box, mapas de sucesión, dashboards de readiness, planes de carrera individuales.
3. **Integraciones**: Legajos, Evaluación (scores), Capacitaciones (planes), Organización (posiciones críticas), Beneficios (programas), Portal Empleado (visibilidad restringida).
4. **Workflow**: aprobación de planes, actualizaciones, revisiones periódicas.
