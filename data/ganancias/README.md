# Tablas oficiales de Ganancias

Completar estos CSV con datos oficiales de ARCA/AFIP y luego convertirlos a JSON con los importadores.

## Archivos
- `art94.csv` (escala)
- `art30.csv` (deducciones personales)

## Importación
Ver `tools/ganancias/README.md`.

## Fuente
Los valores deben tomarse de las tablas oficiales de ARCA/AFIP (Art. 94 y Art. 30).
